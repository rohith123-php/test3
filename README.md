1. Admin Module
The administrative portal serves as the central hub for managing the institution's core data.

Dashboard Analytics: Quick overview of total students, teachers, and active classes.
User Management: Full CRUD (Create, Read, Update, Delete) capabilities to manage Student and Teacher accounts securely.
Academic Setup: Easily add and manage Classes (e.g., 10th Grade) and assign specific Subjects to those classes.
Reporting: View comprehensive school-wide reports, including overall attendance tracking metrics to monitor student engagement.
2. Teacher Module
The teacher portal is designed to reduce paperwork and simplify classroom management.

Classroom Dashboard: View assigned subjects and specialties.
Attendance Management: Digital attendance tracking allowing teachers to mark students as Present or Absent for specific dates, securely saving records to the database.
Academic Grading: A streamlined interface for entering exam marks (e.g., Midterms, Finals) for specific subjects, automatically calculating maximum marks and grades.
3. Student Module
The student portal empowers learners to track their own academic progress independently.

Personal Profile: View personal details, contact information, and assigned class/section.
Attendance Tracking: Real-time access to daily attendance records to track personal presence/absence history.
Academic Results: A transparent view of all exam marks across different subjects, complete with calculated percentages.
Technologies Used
Frontend: HTML5, Vanilla CSS (with custom aesthetics), JavaScript, and Bootstrap 5 for a fully responsive, mobile-friendly design.
Backend: PHP (Procedural/Session-based authentication).
Database: MySQL for structured relational data storage (Tables for Users, Students, Teachers, Classes, Subjects, Attendance, and Marks).
Server Environment: Designed to run seamlessly on local environments like XAMPP, WAMP, or any standard LAMP stack.
